Vroid

VRoid Studio is a 3D modeling software that can create human avatars (roles). The produced 3D module can be output as a usable file through the 3D software. The public beta will only export the VRM format, and will support more file formats in the future. Once you've created a character with VRoid Studio, you can use the character as an avatar on a variety of VR/AR platforms that support the VRM format.

VRoid Provide 3D characters

VRoid is available to all users free of charge. Anyone can create and own their own virtual characters, and the world plan "VRoid Project" that uses 3D virtual characters to create works or communicate with each other in VR/AR space is about to begin.
